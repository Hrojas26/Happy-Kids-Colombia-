# Happy kids Colombia


## install application
 
<ul>
<li> create db</li>
<li> php artisan migrate:fresh</li>
<li> php artisan db:seed</li>

</ul>

## start application
<ul>
<li> Composer install</li>
<li> npm i</li>
<li> npm run dev</li>
<li>php artisan serve</li> 
</ul>
